Equazioni Fattorizzate

10 Esempi risolti applicando la Legge di Annullamento del Prodotto

Regola: Per risolvere queste equazioni, poniamo ogni singola parentesi uguale a zero.
$$(x - 3)(x + 5) = 0$$
Soluzione: \(x = 3 \lor x = -5\)
$$(2x - 1)(x + 4) = 0$$
Soluzione: \(x = \frac{1}{2} \lor x = -4\)
$$x(x - 2)(x + 2) = 0$$
Soluzione: \(x = 0 \lor x = 2 \lor x = -2\)
$$(x + 7)(3x - 2)(x - 1) = 0$$
Soluzione: \(x = -7 \lor x = \frac{2}{3} \lor x = 1\)
$$(x - 10)(x + 10) = 0$$
Soluzione: \(x = 10 \lor x = -10\)
$$(5x + 1)(x - 6) = 0$$
Soluzione: \(x = -\frac{1}{5} \lor x = 6\)
$$(x - 1)(x - 2)(x - 3)(x - 4) = 0$$
Soluzione: \(x = 1 \lor x = 2 \lor x = 3 \lor x = 4\)
$$(4x + 3)(2x - 5) = 0$$
Soluzione: \(x = -\frac{3}{4} \lor x = \frac{5}{2}\)
$$(x + 8)(x - 9)(2x + 1) = 0$$
Soluzione: \(x = -8 \lor x = 9 \lor x = -\frac{1}{2}\)
$$(x + 1)(x - 1)(x + 2)(x - 2) = 0$$
Soluzione: \(x = -1 \lor x = 1 \lor x = -2 \lor x = 2\)

Equazioni da Fattorizzare

Esercizio: Fattorizza le espressioni e trova le soluzioni.
$$x^2 - 9 = 0$$
Differenza di quadrati
Soluzione: \(x = 3 \lor x = -3\)
$$x^2 - 5x = 0$$
Raccoglimento totale
Soluzione: \(x = 0 \lor x = 5\)
$$4x^2 - 1 = 0$$
Differenza di quadrati
Soluzione: \(x = \frac{1}{2} \lor x = -\frac{1}{2}\)
$$3x^2 + 6x = 0$$
Raccoglimento totale
Soluzione: \(x = 0 \lor x = -2\)
$$x^3 - x = 0$$
Raccoglimento + Differenza
Soluzione: \(x = 0 \lor x = 1 \lor x = -1\)
